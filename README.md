# 🫀 Heart Disease Prediction System

This is a Machine Learning project that predicts heart disease risk using a Random Forest Classifier.

## 🚀 Features
- **Professional Dashboard**: Modern UI with real-time feedback.
- **AI Predictions**: Processes patient metrics to determine risk level.
- **Feature Importance**: Analyzes which health factors contribute most to the risk.
- **Kaggle Optimized**: Tailored for the Heart Failure Prediction dataset.

## 🛠️ Tech Stack
- **Backend**: Python, Flask
- **Machine Learning**: Scikit-learn, Pandas, NumPy
- **Frontend**: HTML5, Vanilla CSS3 (Professional Grade)

## ▶️ Setup and Run

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Train the Model** (optional, model.pkl is excluded by gitignore):
   ```bash
   python train_model.py
   ```

3. **Start the Server**:
   ```bash
   python app.py
   ```

4. **Access the Dashboard**:
   Open [http://127.0.0.1:5000](http://127.0.0.1:5000) in your browser.
